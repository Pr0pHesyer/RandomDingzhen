## 随机丁真插件

## 安装
1. 在`HoshinoBot/hoshino/modules`目录clone本项目 
   
   ```
   https://github.com/Pr0pHesyer/MUG_one_sentence.git
   ```
2. 在`HoshinoBot/hoshino/config/__bot__.py`文件的`MODULES_ON`里添加`RandomDingzhen`
3. 重启hoshino

## 指令

| 指令               | 说明                  |
| ---------------- | ------------------- |
| 随机丁真 | 随机获取一张丁真梗图 |
